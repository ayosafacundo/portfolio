# Personal Portfolio

Frontend made with Replit Design AI, deployed to my own Homelab through Portainer, with Nginx as reverse proxy setup, through Cloudflare for easy DNS configuration and security. Expressjs backend with postgresql database connection, and gotify for push notifications.

## You can see the deployed project here:

https://ayosafacundo.com.ar

### Tech Stack:

- Typescript
- Vite
- Reactjs
- Expressjs
- Tailwindcss
- Nodejs
- Postgresql
